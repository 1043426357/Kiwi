Kiwi

Kiwi makes Behavior Driven Development for iOS and OS X development simply pleasurable.

Why?

The idea behind Kiwi is to have tests that are more readable than what is possible with the test framework bundled with Xcode.

Tests (formally Specs) are written in Objective-C and are run within the comfort of Xcode to provide a test environment that is unobtrusive and seamless.

Specs look like this:

describe(@"Team", ^{
    context(@"when newly created", ^{
        it(@"should have a name", ^{
            id team = [Team team];
            [[team.name should] equal:@"Black Hawks"];
        });

        it(@"should have 11 players", ^{
            id team = [Team team];
            [[[team should] have:11] players];
        });
    });
});
